# Système CSS

## Format

Travail individuel

## Consigne du projet

Créez un mini-système CSS pour un site de portfolio avec :

- 3 composants minimum (ex: header, project-card, button)
- Variables CSS pour couleurs, espacements, typographie
- Nomenclature BEM cohérente
- Documentation complète

## Livrables attendus

1. Fichier `system.css` avec :
    - Variables CSS (`:root`)
    - Au moins 3 composants documentés
    - Nomenclature BEM
2. Fichier `index.html` qui utilise les composants
3. Fichier `DOCUMENTATION.md` qui documente :
    - Liste des composants
    - Convention de nommage utilisée
    - Comment utiliser le système
